# ⚡ Portfolio — DEV.EXE (Cyber-Terminal Edition)

Un portfolio immersif à l'esthétique **Retro-Futuriste / Cyberpunk**, conçu comme une interface de terminal haute performance. Ce projet met en avant des projets de développement web avec une expérience utilisateur (UX) interactive et visuelle.

## 🔴 Aperçu du Design
- **Ambiance CRT/VHS** : Effets de lignes de balayage (scanlines), vignette et animations de scintillement (flicker).
- **Interactivité** : Curseur personnalisé dynamique, effet de parallaxe (Tilt) sur les cartes de projets et compteurs animés.
- **Typographie** : Mélange de `VT323` (code), `Special Elite` (machine à écrire) et `Libre Baskerville`.

## 🛠️ Stack Technique
Ce projet est une démonstration de maîtrise du **Frontend Vanilla** (sans frameworks lourds) pour une performance optimale :
- **Langage** : HTML5, CSS3 (Variables personnalisées, Flexbox, Grid).
- **Animations & Logique** : JavaScript ES6+.
- **APIs Web** : 
  - `IntersectionObserver` pour les animations au défilement.
  - `RequestAnimationFrame` pour la fluidité des compteurs.

## ✨ Fonctionnalités Clés
1. **Système de Filtrage Dynamique** : Tri des projets par catégorie (Vitrine, Web App) sans rechargement de page.
2. **Effet Tilt 3D** : Les cartes de projets réagissent au mouvement de la souris (perspective CSS).
3. **Statistiques Animées** : Des compteurs (Projets, Clients, Expérience) s'activent dès qu'ils entrent dans le champ de vision.
4. **Interface Immersive** : Navigation avec effets de flou (backdrop-filter) et curseur "traqueur" personnalisé.
